# Traductor de Portapapeles

Esto es una herramienta que traduce automáticamente cualquier texto copiado de manera offline.

Nota: Como dije, esto es una IA corriendo en local. Está tan optimizado como me fue posible, pero hay computadoras en las cuales el rendimiento puede ser muy bajo.

## ¿Cómo usarlo?
1. Ejecutá el archivo `Traductor de Portapapeles.exe`.
2. Escucharás el aviso "Traductor activado".
3. Copiá algún texto, y esperá la traducción.

Para cambiar los idiomas, haz doble clic en el icono de la bandeja o presiona `Ctrl + Shift + C`.

## Modificación de Idiomas
Actualmente el sistema soporta más de 60 idiomas (Español, Inglés, Irlandés, Portugués, Francés, Alemán, etc.). 
Por defecto, el Idioma de Origen es Inglés y el Destino es Español.

Nota: asegurate de ser preciso con el idioma de origen, o la IA podría alucinar.
