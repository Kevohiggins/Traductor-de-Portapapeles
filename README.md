# Traductor de Portapapeles

Este programa intercepta cualquier texto que copies al portapapeles y lo traduce instantáneamente al idioma de tu elección, leyéndolo en voz alta.

## Modelos de IA Disponibles

El programa viene preconfigurado con una selección de modelos que, en teoría, deberían funcionar gratuitamente:

- **OpenAI GPT-OSS 120B**.
- **NVIDIA Nemotron 3 Ultra 550B**.
- **Llama 4 Scout / Llama 3.1 8B**.
- **Qwen 3 30B / Mistral NeMo**:.
- **Google Gemma 4 31B**.

## Atajos de Teclado Globales

- `Control + Shift + C`: Abre la ventana de **Ajustes**.
- `Control + Shift + Suprimir: **Vacía el historial** de memoria de la IA (ideal cuando cambias de juego).
- `Control + Shift + F4`: **Cierra el programa**.

## Cómo obtener tu clave de OpenRouter

Para que el programa funcione, necesitás una clave API de OpenRouter. Es un proceso de 1 minuto:

1. Entrá a [la página de open Router](https://openrouter.ai/) y creá una cuenta (podés usar Google o GitHub).
2. Andá a tu perfil y buscá la sección **"Keys"** (Claves).
3. Entrá en **"Create Key"**, ponele el nombre que quieras (ej. "Traductor") y copiala.
4. Pegá esa clave en los ajustes del programa o, preferentemente, en la primera ejecución ah.

## Modo Contextual

En los Ajustes podés activar el **Modo Contextual**. En este modo, se va guardando la charla actual. Sirve para traducir juegos, por ejemplo, porque la IA va recordando los nombres, cómo es el lore... todo para que fluya mejor la traducción, obviamente.

Podés usar `Control + Shift + Suprimir` cuando quieras limpiar esa memoria para arrancar un texto nuevo. Planeo implementar, a futuro, alguna manera de guardar las conversaciones e ir alternando para no borrarlas a fuerzas.
