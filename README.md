# Traductor Gemma 3

Esto es una herramienta que traduce automáticamente cualquier texto copiado de manera offline y gratuita, protegiendo tu privacidad.

Nota: Como dije, esto es una IA corriendo en local. Está tan optimizado como me fue posible usando CTranslate2, pero hay computadoras en las cuales el rendimiento puede ser bajo debido a que el modelo (Gemma 3) requiere memoria RAM/VRAM decente.

## ¿Cómo usarlo?
1. Ejecutá el archivo `Traductor Gemma.exe`.
2. Escucharás el aviso "Traductor activado".
3. Copiá algún texto, y esperá la traducción.

Para pausar temporalmente, presiona `Ctrl + Shift + A`.
Para cambiar los idiomas, haz doble clic en el icono de la bandeja o presiona `Ctrl + Shift + C`.

## Modificación de Idiomas
Actualmente el sistema soporta muchísimos idiomas gracias al modelo de Google. 
Por defecto, el Idioma de Origen es Inglés y el Destino es Español.

## Novedades V3
Esta versión utiliza **Google Gemma 3**, por lo que ya no hace traducciones literales máquina-a-máquina. Entiende contexto, dobles sentidos, dichos, chistes y metáforas como un ser humano. 
Además incluye una memoria de contexto opcional para que recuerde tus traducciones anteriores.

Nota: asegurate de ser preciso con el idioma de origen, o la IA podría alucinar.
